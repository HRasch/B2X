# Wie man sein Passwort zurückgesetzt

Passwort vergessen? Kein Problem! Sie können es in wenigen Minuten zurücksetzen.

---

## Schnelle Schritte

### Schritt 1: Gehen Sie zur Anmeldeseite
1. Gehen Sie zu **www.B2X.com**
2. Klicken Sie auf die **"Anmelden"**-Schaltfläche

### Schritt 2: Klicken Sie auf "Passwort vergessen?"
1. Klicken Sie auf der Anmeldeseite auf den Link **"Passwort vergessen?"**
2. Die Seite zum Zurücksetzen des Passworts öffnet sich

### Schritt 3: Geben Sie Ihre E-Mail ein
1. Geben Sie die E-Mail-Adresse für Ihr Konto ein
2. Klicken Sie auf **"Zurücksetzen-Link senden"**
3. Sie sehen: "Überprüfen Sie Ihre E-Mail auf Anweisungen zum Zurücksetzen"

### Schritt 4: Überprüfen Sie Ihre E-Mail
1. Öffnen Sie Ihre E-Mail (Gmail, Outlook usw.)
2. Suchen Sie nach einer E-Mail von **noreply@B2X.com**
3. Betreff: **"Setzen Sie Ihr B2X-Passwort zurück"**
4. **Sehen Sie es nicht?** Überprüfen Sie den Spam/Junk-Ordner

### Schritt 5: Klicken Sie auf den Zurücksetzen-Link
1. Öffnen Sie die E-Mail von B2X
2. Klicken Sie auf die Schaltfläche **"Passwort zurücksetzen"** (oder kopieren Sie den Link)
3. Die Zurücksetzen-Seite öffnet sich (Link funktioniert nur 1 Stunde)

### Schritt 6: Erstellen Sie ein neues Passwort
1. Geben Sie Ihr neues Passwort ein
2. Geben Sie es zur Bestätigung erneut ein
3. **Passwortanforderungen:**
   - Mindestens 8 Zeichen
   - Gemisch aus Großbuchstaben UND Kleinbuchstaben
   - Mindestens eine Zahl
   - Mindestens ein Sonderzeichen (!@#$%^&*)
4. Klicken Sie auf **"Neues Passwort speichern"**

### Schritt 7: Melden Sie sich an
1. Gehen Sie zur Anmeldeseite
2. Geben Sie Ihre E-Mail und Ihr neues Passwort ein
3. Klicken Sie auf **"Anmelden"**
4. Willkommen zurück! 🎉

## Passwort-Tipps

### Was macht ein gutes Passwort?

✅ **GUT:**
- `Sommer@Paris2025`
- `Blau#Berge123`
- `Kaffee&Kuchen!456`

❌ **SCHLECHT:**
- `password123` (zu häufig)
- `12345678` (nur Zahlen)
- `ABCDEFGH` (nur Großbuchstaben)
- `meinname` (zu einfach)

### Wie man sein Passwort merkt

1. **Verwenden Sie einen Satz**: Denken Sie an einen einprägsamen Satz
   - Satz: "Ich liebe Kaffee um 7 Uhr morgens"
   - Passwort: `IlKu7Um!`

2. **Verwenden Sie ein Muster**: Tastaturbewegung + Zahlen
   - Muster: `QwErTy2025!`

3. **Verwenden Sie einen Passwort-Manager**: Werkzeuge wie LastPass oder 1Password speichern Passwörter sicher

## Fehlerbehebung

### "E-Mail nicht gefunden"

**Was bedeutet das?**  
Bei B2X existiert kein Konto mit dieser E-Mail-Adresse.

**Wie man es behebt:**
1. Überprüfen Sie die Schreibweise der E-Mail
2. Versuchen Sie eine andere E-Mail, die Sie möglicherweise verwendet haben
3. [Erstellen Sie ein neues Konto](./getting-started.md), wenn Sie noch kein Konto haben

### "Zurücksetzen-Link abgelaufen"

**Was bedeutet das?**  
Der Zurücksetzen-Link funktioniert nur 1 Stunde. Sie haben zu lange gewartet.

**Wie man es behebt:**
1. Gehen Sie zurück zur [Seite zum Zurücksetzen des Passworts](https://www.B2X.com/forgot-password)
2. Geben Sie Ihre E-Mail erneut ein
3. Klicken Sie auf den neuen Link

### "E-Mail nicht empfangen"

**Überprüfen Sie diese Orte:**
1. ✓ Überprüfen Sie den Spam-Ordner
2. ✓ Überprüfen Sie den Promotions-Ordner (Gmail)
3. ✓ Überprüfen Sie den Junk-Ordner (Outlook)
4. ✓ Warten Sie 5 Minuten (E-Mails können langsam sein)
5. ✓ Überprüfen Sie die Schreibweise Ihrer E-Mail

**Immer noch nichts?**
1. Gehen Sie zurück zur Zurücksetzen-Seite
2. Geben Sie Ihre E-Mail erneut ein
3. Überprüfen Sie einen anderen E-Mail-Ordner

### "Passwort erfüllt die Anforderungen nicht"

**Was bedeutet das?**  
Ihrem Passwort fehlt etwas Erforderliches.

**Anforderungen:**
- ✓ Mindestens 8 Zeichen lang
- ✓ Mindestens ein GROSSBUCHSTABE (A-Z)
- ✓ Mindestens ein Kleinbuchstabe (a-z)
- ✓ Mindestens eine Zahl (0-9)
- ✓ Mindestens ein Symbol (!@#$%^&*)

**Beispiel, das funktioniert:**
- `MeinNeuesPasswort2025!`

## Häufig gestellte Fragen

**F: Wie lange dauert das Passwort-Zurücksetzen per E-Mail?**  
A: Normalerweise weniger als 1 Minute. Überprüfen Sie den Spam-Ordner, wenn Sie die E-Mail nicht sehen.

**F: Kann ich mein Passwort in der mobilen App zurücksetzen?**  
A: Ja! Suchen Sie auf dem Anmeldebildschirm nach „Passwort vergessen?".

**F: Was ist, wenn ich keinen Zugriff auf die E-Mail in meinem Konto habe?**  
A: [Kontaktieren Sie den Support](../README.md) mit Eigentumsnachweis.

**F: Muss ich mein Passwort regelmäßig zurücksetzen?**  
A: Nein, es sei denn, Sie denken, dass Ihr Konto kompromittiert ist.

**F: Kann ich ein altes Passwort wiederverwenden?**  
A: Nein, aus Sicherheitsgründen lässt B2X Sie alte Passwörter nicht wiederverwenden.

## Sicherheits-Erinnerung

⚠️ **Setzen Sie Ihr Passwort niemals über einen E-Mail-Link zurück, es sei denn:**
1. Sie haben selbst auf "Passwort vergessen?" geklickt
2. Die E-Mail ist von `noreply@B2X.com`
3. Die URL beginnt mit `https://www.B2X.com`

Hacker versenden manchmal gefälschte Passwort-Zurücksetzen-E-Mails. Starten Sie immer von der offiziellen Website.
