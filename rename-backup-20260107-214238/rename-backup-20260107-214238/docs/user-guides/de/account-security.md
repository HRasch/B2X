# Kontotipps zur Sicherheit

Halten Sie Ihr B2X-Konto sicher! Befolgen Sie diese einfachen Schritte.

---

## Wesentliche Sicherheitsregeln

### 1. Verwenden Sie ein sicheres Passwort

✅ **TUN Sie:**
- Mischen Sie Großbuchstaben und Kleinbuchstaben: `MeinPasswort`
- Zahlen einbeziehen: `MeinPasswort123`
- Sonderzeichen hinzufügen: `MeinPasswort123!`
- Verwenden Sie mindestens 8 Zeichen

❌ **TUN Sie NICHT:**
- Verwenden Sie Ihren Namen oder Geburtstag
- Verwenden Sie häufige Wörter wie "passwort"
- Verwenden Sie überall das gleiche Passwort
- Teilen Sie Ihr Passwort mit jemandem

### 2. Teilen Sie niemals Ihr Passwort

**Ihr Passwort sollte nur Ihnen bekannt sein!**

Wer sollte Ihr Passwort NICHT haben:
- ❌ Freunde und Familie
- ❌ B2X-Mitarbeiter (Sie werden es niemals fragen!)
- ❌ Ihr Ehepartner oder Partner
- ❌ Kundensupport-Personal

**Ausnahme:** Passwort-Manager (LastPass, 1Password) - sie sind verschlüsselt und sicher.

### 3. Seien Sie vorsichtig auf öffentlichen Computern

**An einem öffentlichen Computer (Bibliothek, Hotel, Büro):**
1. ❌ Verwenden Sie nicht „Angemeldet bleiben"
2. ❌ Speichern Sie Ihr Passwort nicht
3. ✅ Klicken Sie immer auf „Abmelden", wenn Sie fertig sind
4. ✅ Schließen Sie das Browserfenster vollständig

**Zu Hause (Ihr Computer):**
- ✅ Es ist sicherer, „Angemeldet bleiben" zu verwenden

### 4. Verwenden Sie nur das offizielle B2X

✅ **SICHER:**
- Website: `https://www.B2X.com`
- Offizielle mobile App (iOS oder Android)
- Offizielle B2X-E-Mails (von `@B2X.com`)

❌ **NICHT SICHER:**
- Websites, die wie B2X aussehen, aber eine andere URL haben
- Unbekannte Apps, die behaupten, B2X zu sein
- E-Mails, die Sie auffordern, Ihr Passwort zu "bestätigen"

### 5. Überprüfen Sie, ob E-Mails echt sind

**Betrüger versenden gefälschte E-Mails!** So erkennen Sie sie:

✅ **Echte B2X-E-Mails:**
- Kommen von `@B2X.com` oder `noreply@B2X.com`
- Behandeln Sie als "[Ihr Name]"
- Handeln von Dingen, die Sie getan haben (Bestellungen, Anmeldung usw.)

❌ **Gefälschte E-Mails:**
- Kommen von `@gmail.com` oder unbekannten Domänen
- Sagen "Lieber Kunde" (nicht Ihr Name)
- Bitten Sie, "Ihr Konto zu verifizieren" oder "Zahlung zu bestätigen"
- Enthalten verdächtige Links
- Fragen Sie nach Ihrem Passwort (B2X fragt NIEMALS danach!)

**Im Zweifelsfall:** Gehen Sie direkt zu B2X.com, anstatt auf E-Mail-Links zu klicken.

## Zwei-Faktor-Authentifizierung (2FA)

### Was ist 2FA?

Die Zwei-Faktor-Authentifizierung fügt eine zusätzliche Sicherheitsebene hinzu:
1. Etwas, das Sie wissen: **Ihr Passwort**
2. Etwas, das Sie haben: **Ihr Telefon**

### Wie es funktioniert

```
Sie melden sich mit E-Mail und Passwort an
        ↓
B2X sendet einen Code an Ihr Telefon
        ↓
Sie geben den Code ein
        ↓
Sie sind angemeldet! ✓
```

Selbst wenn jemand Ihr Passwort stiehlt, können er sich nicht ohne Ihr Telefon anmelden!

### Wie man 2FA aktiviert

1. Melden Sie sich bei Ihrem Konto an
2. Gehen Sie zu **Einstellungen** → **Sicherheit**
3. Klicken Sie auf **"Zwei-Faktor-Authentifizierung aktivieren"**
4. Wählen Sie Versandmethode:
   - **SMS (Textnachricht)** - Code wird per Text an Ihr Telefon gesendet
   - **Authenticator-App** - Verwenden Sie Google Authenticator oder Microsoft Authenticator
5. Führen Sie die Setup-Schritte aus
6. Speichern Sie Sicherungscodes an einem sicheren Ort!
7. Klicken Sie auf **"Aktivieren"**

### Verwenden von 2FA beim Anmelden

**Mit SMS:**
1. Geben Sie E-Mail und Passwort ein
2. Textnachricht mit 6-stelligem Code erhalten
3. Geben Sie den Code auf der Anmeldeseite ein
4. Sie sind drin!

**Mit Authenticator-App:**
1. Geben Sie E-Mail und Passwort ein
2. Öffnen Sie Google Authenticator oder Authy-App
3. Finden Sie B2X-Code (ändert sich alle 30 Sekunden)
4. Geben Sie den Code auf der Anmeldeseite ein
5. Sie sind drin!

### Sicherungscodes

Wenn Sie 2FA aktivieren, erhalten Sie Sicherungscodes. **Speichern Sie sie an einem sicheren Ort!**

**Warum?** Wenn Sie Ihr Telefon verlieren, können Sie Sicherungscodes verwenden, um sich anzumelden.

**Wo speichern:**
- ✅ Auf Papier in einem Safe schreiben
- ✅ Verschlüsselter Passwort-Manager
- ❌ Nicht in Ihrer E-Mail

## Was ist, wenn...

### Sie denken, Ihr Passwort ist kompromittiert

1. **Sofort** gehen Sie zu [Setzen Sie Ihr Passwort zurück](./reset-password.md)
2. Erstellen Sie ein völlig neues Passwort
3. Überprüfen Sie Ihre letzten Bestellungen (sehen Sie, ob etwas verdächtig ist)
4. [Kontaktieren Sie den Support](../README.md), wenn Sie ungewöhnliche Aktivität sehen

### Sie haben Ihr Telefon verloren (und haben 2FA aktiviert)

1. Gehen Sie zur B2X-Anmeldeseite
2. Klicken Sie auf "Kann nicht auf meinen 2FA-Code zugreifen?"
3. Geben Sie Ihre E-Mail ein
4. Verwenden Sie einen Ihrer Sicherungscodes
5. Melden Sie sich an
6. Deaktivieren Sie 2FA oder verknüpfen Sie ein neues Telefon
7. [Kontaktieren Sie den Support](../README.md), wenn Sie Sicherungscodes verloren haben

### Sie sehen verdächtige Anmeldeaktivität

1. **Ändern Sie Ihr Passwort sofort**
2. Überprüfen Sie aktuelle Bestellungen und Kontoaktivität
3. Überprüfen Sie, ob Adressen oder Zahlungsmethoden geändert wurden
4. [Kontaktieren Sie den Support](../README.md) sofort

### Jemand anders hat Zugriff auf Ihre E-Mail

Wenn Ihr E-Mail-Konto kompromittiert ist:

1. **Ändern Sie das E-Mail-Passwort sofort** (auf Gmail, Outlook usw.)
2. [Setzen Sie Ihr B2X-Passwort zurück](./reset-password.md)
3. Überprüfen Sie Ihre E-Mail-Weiterleitungsregeln (Hacker fügen diese manchmal hinzu)
4. Aktivieren Sie 2FA auf Ihrem E-Mail-Konto
5. [Kontaktieren Sie den Support](../README.md), wenn nötig

## Passwort-Tipps und Tricks

### Wie man sichere Passwörter erstellt

**Methode 1: Persönliche Sätze**
- Satz: "Mein erster Hund hieß Buddy"
- Passwort: `MehHBuddy2025!`

**Methode 2: Zufälliges Gemisch**
- Großbuchstaben: A, R, T
- Kleinbuchstaben: m, o, w
- Zahlen: 8, 3
- Symbole: !
- Ergebnis: `ArtmowB!83` (dann mischen)

**Methode 3: Einprägsamer Satz**
- Satz: "Ich trinke jeden Tag um 7 Uhr morgens Kaffee"
- Passwort: `ItJdU7UmMK!`

### Verwenden eines Passwort-Managers

Passwort-Manager speichern Ihre Passwörter sicher:
- ✅ LastPass
- ✅ 1Password
- ✅ Bitwarden (kostenlos)
- ✅ KeePass (kostenlos)

**Vorteile:**
- Generieren Sie sichere Zufallspasswörter
- Vergessen Sie niemals Passwörter
- Sicher auf all Ihren Geräten
- Verschlüsselte Lagerung

## Häufig gestellte Fragen

**F: Wie oft sollte ich mein Passwort ändern?**  
A: Nur wenn Sie denken, dass es kompromittiert ist. Das häufige Ändern von Passwörtern ist nicht erforderlich, wenn Sie ein sicheres Passwort verwenden.

**F: Ist B2X sicher zum Einkaufen?**  
A: Ja! Wir verwenden Sicherheitsstandards der Branche (HTTPS, Verschlüsselung). Ihre Zahlungsinformationen werden von sicheren Zahlungsanbietern verarbeitet.
**F: Verkaufen Sie meine persönlichen Daten?**  
A: Niemals! Ihre Daten werden nur für Bestellungen und Kundendienst verwendet.
**F: Wie lösche ich mein Konto?**  
A: Gehen Sie zu Einstellungen → Konto → Konto löschen. [Kontaktieren Sie den Support](../README.md) für Hilfe.
**F: Was sollte ich tun, wenn ich Gebühren sehe, die ich nicht gemacht habe?**  
A: [Kontaktieren Sie den Support](../README.md) sofort. Wir untersuchen betrügerische Gebühren.
