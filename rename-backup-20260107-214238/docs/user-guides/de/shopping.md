# Einkaufen bei B2X

Willkommen! Diese Anleitung zeigt Ihnen, wie Sie Produkte finden, in den Warenkorb legen und bestellen können.

---

## 🔍 Produkte finden

### Stöbern im Katalog

1. **Kategorien durchsuchen**
   - Klicken Sie auf eine Kategorie in der Navigation
   - Unterkategorien werden angezeigt

2. **Produktliste**
   - Produkte werden als Karten angezeigt
   - Jede Karte zeigt: Bild, Name, Preis, Verfügbarkeit

### Suche verwenden

1. **Suchfeld**
   - Geben Sie einen Suchbegriff ein (z.B. Produktname, SKU)
   - Drücken Sie Enter oder klicken Sie auf die Lupe

2. **Suchergebnisse**
   - Relevante Produkte werden angezeigt
   - Sortieren Sie nach: Relevanz, Preis, Name

3. **Filter anwenden**
   - Kategorie auswählen
   - Preisbereich eingeben
   - Marke auswählen

---

## 🛒 Warenkorb

### Produkt hinzufügen

1. Klicken Sie auf ein Produkt für Details
2. Wählen Sie die gewünschte Menge
3. Klicken Sie auf **"In den Warenkorb"**

> ✅ Eine Bestätigung erscheint oben rechts

### Warenkorb anzeigen

1. Klicken Sie auf das Warenkorb-Symbol (🛒) oben rechts
2. Der Warenkorb öffnet sich als Seitenleiste

### Menge ändern

- **+** / **-** Buttons neben jedem Artikel
- Oder: Menge direkt eingeben

### Artikel entfernen

- Klicken Sie auf das **X** neben dem Artikel
- Bestätigen Sie die Entfernung

---

## 📦 Versandkosten

Bevor Sie zur Kasse gehen, sehen Sie die Versandkosten:

1. **Lieferland wählen**
   - Wählen Sie Ihr Land aus dem Dropdown
   
2. **Versandart wählen**
   - Standardversand (3-5 Tage)
   - Expressversand (1-2 Tage)
   - Abholung (wenn verfügbar)

3. **Preisanzeige**
   - Zwischensumme (Produkte)
   - + Versandkosten
   - = **Gesamtpreis inkl. MwSt.**

> ℹ️ Alle Preise werden gemäß PAngV transparent angezeigt

## ❓ Häufige Fragen

**F: Kann ich ohne Konto einkaufen?**  
A: Zum Stöbern ja, aber für Bestellungen benötigen Sie ein Konto.

**F: Wie lange bleiben Artikel im Warenkorb?**  
A: Ihr Warenkorb wird 30 Tage gespeichert (bei angemeldetem Konto).

**F: Kann ich den Warenkorb speichern?**  
A: Ja, melden Sie sich an und Ihr Warenkorb wird automatisch gespeichert.

**F: Warum sehe ich unterschiedliche Preise?**  
A: B2B-Kunden sehen Nettopreise, B2C-Kunden sehen Bruttopreise inkl. MwSt.
