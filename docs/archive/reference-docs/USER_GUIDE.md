## 🛍️ Für Kunden (Store)

### Getting Started - Erste Schritte

#### 1. Konto erstellen

1. Rufen Sie die **B2X Store** auf
2. Klicken Sie auf **"Registrieren"** oder **"Sign Up"**
3. Geben Sie ein:
   - **E-Mail-Adresse** (für Benachrichtigungen)
   - **Passwort** (Mindestens 8 Zeichen, mit Großbuchstaben & Zahlen)
   - **Vorname & Nachname**
4. Akzeptieren Sie die **Nutzungsbedingungen**
5. Klicken Sie **"Konto erstellen"**
6. **Bestätigen Sie Ihre E-Mail** (Link in der Bestätigungs-E-Mail)

✅ **Konto ist jetzt aktiv!**

---

#### 2. Anmelden

1. Klicken Sie auf **"Anmelden"** oder **"Login"**
2. Geben Sie **E-Mail** und **Passwort** ein
3. Klicken Sie **"Anmelden"**

**Optional:** Aktivieren Sie **"Angemeldet bleiben"** zum automatischen Login

---

#### 3. Passwort vergessen?

1. Auf der Login-Seite: Klicken Sie **"Passwort vergessen?"**
2. Geben Sie Ihre **E-Mail-Adresse** ein
3. Klicken Sie **"Passwort zurücksetzen"**
4. **Öffnen Sie Ihre E-Mail** und klicken Sie den Reset-Link
5. Geben Sie ein **neues Passwort** ein
6. Klicken Sie **"Passwort aktualisieren"**

---

### Einkaufen

#### 4. Produkte durchsuchen

**Option A: Nach Kategorie**
```
Home → Kategorien → Wählen Sie eine Kategorie
→ Alle Produkte in dieser Kategorie werden angezeigt
```

**Option B: Mit Suchfunktion**
```
Suchleiste oben → Geben Sie "Produktname" ein
→ Relevante Produkte werden angezeigt
```

**Option C: Nach Filter**
```
Linke Seite → Filter anwenden:
  - Preis: Min - Max eingeben
  - Verfügbarkeit: "Verfügbar" wählen
  - Sortierung: "Beste Bewertung" o.ä.
```

---

#### 5. Produkt anschauen

Klicken Sie auf ein **Produkt-Thumbnail**:

**Auf dieser Seite sehen Sie:**
- 📸 **Produktbilder** (Klicken zum Vergrößern)
- 📝 **Produktbeschreibung**
- ⭐ **Kundenbewertungen**
- 💰 **Preis & Verfügbarkeit**
- 📦 **Größe/Varianten** (falls vorhanden)
- 🛒 **"In den Warenkorb" Button**

---

#### 6. Warenkorb verwenden

**In den Warenkorb legen:**
```
1. Wählen Sie Varianten (Größe, Farbe, etc.) falls nötig
2. Geben Sie Menge ein
3. Klicken Sie "In den Warenkorb"
```

**Warenkorb anschauen:**
```
Oben rechts: Klicken Sie auf 🛒 "Warenkorb"
Hier sehen Sie:
  - Alle Artikel in Ihrem Warenkorb
  - Preise & Mengen
  - Zwischensumme
```

**Artikel entfernen:**
```
Im Warenkorb → Neben jedem Artikel → Klicken Sie ❌
```

---

#### 7. Zur Kasse gehen (Checkout)

```
Warenkorb → Klicken Sie "Zur Kasse"
```

**Schritt 1: Lieferdaten**
- Lieferadresse eingeben oder auswählen
- Lieferart wählen (Standard, Express, etc.)

**Schritt 2: Zahlungsmethode**
- Kreditkarte, PayPal, oder andere Methoden
- Zahlungsdaten eingeben

**Schritt 3: Bestellübersicht**
- Alle Details überprüfen
- Klicken Sie "Bestellung bestätigen"

✅ **Bestellung erfolgreich!**
- Sie erhalten eine **Bestätigungs-E-Mail**
- Sie können den Status unter **"Meine Bestellungen"** verfolgen

---

### Mein Konto

#### 8. Profil anschauen & ändern

```
Oben rechts: Klicken Sie auf 👤 "Mein Konto"
→ "Mein Profil"
```

**Ändern Sie:**
- ✏️ Namen

- ✏️ Telefonnummer
- ✏️ Profilbild

**Speichern:** Klicken Sie **"Änderungen speichern"**

---

#### 9. Meine Bestellungen

```
Mein Konto → "Bestellungen"
```

**Für jede Bestellung sehen Sie:**
- 📦 Bestellnummer
- 📅 Bestelldatum
- 💰 Gesamtpreis
- 🚚 Versandstatus
- ⏱️ Geschätztes Lieferdatum

**Bestellung anschauen:**

- Sehen Sie Artikel, Adresse, Zahlungsmethod e

#### 10. Adressbuch

```
Mein Konto → "Adressen"
```

**Neue Adresse hinzufügen:**
```
1. Klicken Sie "+ Neue Adresse"
2. Geben Sie ein:
   - Straße & Hausnummer
   - Postleitzahl & Stadt
   - Land
3. Klicken Sie "Speichern"
```

**Adresse beim Checkout verwenden:**
```
Bei "Lieferdaten" → Wählen Sie eine gespeicherte Adresse
```

---

#### 11. Passwort ändern

```
Mein Konto → "Sicherheit"
→ "Passwort ändern"
```

**Geben Sie ein:**
1. **Aktuelles Passwort** (Bestätigung)
2. **Neues Passwort**
3. **Neues Passwort wiederholen**

Klicken Sie **"Passwort ändern"**

## ❓ Häufig gestellte Fragen (FAQ)

### Für Kunden

**F: Wie lange dauert der Versand?**
A: Der Versand dauert normalerweise 3-5 Werktage. Mit Express-Versand 1-2 Werktage.

**F: Kann ich eine Bestellung stornieren?**
A: Ja, wenn die Bestellung noch nicht versendet wurde. Rufen Sie den Support an: support@B2X.com

**F: Wie kann ich ein Produkt zurückgeben?**
A: Innerhalb von 30 Tagen nach Kauf. Rufen Sie Support an für ein Rückgabeformular.

**F: Sind meine Zahlungsdaten sicher?**
A: Ja! Wir verschlüsseln alle Zahlungsdaten und speichern sie nicht auf unseren Servern.

**F: Kann ich meine Bestellung nachverfolgen?**
A: Ja, unter "Meine Bestellungen" → Klicken Sie "Tracking".

---

### Für Admin

**F: Wie ändere ich den Produktpreis?**
A: Produkte → Wählen Sie Produkt → Bearbeiten → Preis ändern → Speichern

**F: Wie kann ich mehrere Bestellungen gleichzeitig exportieren?**
A: Bestellungen → Wählen Sie mehrere Bestellungen (Checkboxen) → "Exportieren"

**F: Wie setze ich ein Passwort für einen Benutzer zurück?**
A: Benutzer → Wählen Sie Benutzer → "Passwort zurücksetzen" → Neues Passwort wird per E-Mail gesendet

**F: Wo finde ich die Verkaufsstatistiken?**
A: Reports → Analytics → Wählen Sie Report-Typ → Filtern Sie nach Datum/Kategorie
