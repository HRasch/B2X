# Dashboard – Admin Guide

> **Modul**: Dashboard  
> **Zielgruppe**: Alle Admin-Benutzer  
> **Version**: 1.0

---

## Übersicht

Das Dashboard ist die Startseite des Admin Portals. Es bietet einen schnellen Überblick über die wichtigsten Kennzahlen und Aktivitäten.

## Bereiche

### 1. Statistik-Karten

Im oberen Bereich finden Sie vier Statistik-Karten:

| Karte | Beschreibung |
|-------|--------------|
| **Gesamtumsatz** | Umsatz im aktuellen Monat |
| **Neue Bestellungen** | Anzahl der Bestellungen heute |
| **Neue Kunden** | Registrierungen diese Woche |
| **Lagerbestand** | Produkte mit niedrigem Bestand |

### 2. Verkaufsübersicht

Das Diagramm zeigt:
- **Zeitraum**: Letzte 7 Tage
- **Daten**: Täglicher Umsatz
- **Interaktion**: Hover für Details

### 3. Letzte Aktivitäten

Zeigt die neuesten Aktionen:
- ✅ **Erfolgreiche Aktionen** (grün)
- ⚠️ **Warnungen** (gelb)
- ❌ **Fehler** (rot)

Jeder Eintrag enthält:
- Benutzer-Avatar
- Aktionsbeschreibung
- Zeitstempel
- Status-Badge

### 4. Neue Benutzer

Tabelle mit den neuesten Registrierungen:
- Name und Avatar
- E-Mail-Adresse
- Kontoststatus (Aktiv/Ausstehend/Inaktiv)
- Registrierungsdatum

## Schnellaktionen

| Button | Aktion |
|--------|--------|
| **Create New** | Neues Element erstellen |
| **View All** | Zur Übersichtsliste |
| **More Options** | Erweiterte Optionen |

## Tastaturkürzel

| Kürzel | Aktion |
|--------|--------|
| `G` + `D` | Zum Dashboard |
| `G` + `P` | Zu Produkten |
| `G` + `U` | Zu Benutzern |
| `?` | Hilfe anzeigen |

## Häufige Fragen

### Wie aktualisiere ich die Statistiken?

Die Statistiken werden automatisch alle 5 Minuten aktualisiert. Für manuelle Aktualisierung drücken Sie `F5` oder klicken Sie auf das Refresh-Icon.

### Warum sehe ich keine Aktivitäten?

Mögliche Ursachen:
1. **Keine Aktivität**: Es gab heute noch keine Aktionen
2. **Berechtigung**: Sie haben keinen Zugriff auf alle Aktivitäten
3. **Filter aktiv**: Prüfen Sie die Filtereinstellungen
