## Übersicht

Das Admin Portal ist die zentrale Verwaltungsoberfläche für die B2X B2B-Plattform. Hier verwalten Administratoren Produkte, Kategorien, Benutzer und CMS-Inhalte.

## Erste Schritte

### 1. Anmeldung

1. Navigieren Sie zu `https://admin.ihre-domain.de`
2. Geben Sie Ihre E-Mail-Adresse und Passwort ein
3. Klicken Sie auf **Anmelden**

### 2. Dashboard

Nach der Anmeldung sehen Sie das Dashboard mit:
- **Statistiken**: Verkaufszahlen, Benutzerregistrierungen
- **Aktivitäten**: Letzte Aktionen im System
- **Schnellzugriff**: Wichtige Funktionen

## Berechtigungen

| Rolle | Beschreibung | Berechtigungen |
|-------|--------------|----------------|
| **Super Admin** | Vollzugriff | Alle Funktionen |
| **Admin** | Standard-Admin | Produkte, Kategorien, CMS |
| **User Manager** | Benutzerverwaltung | Nur Benutzerverwaltung |
| **Content Editor** | CMS-Zugriff | Nur CMS-Funktionen |
