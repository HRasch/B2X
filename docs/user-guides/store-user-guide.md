# User Guide: B2Connect Store Interface

**Audience:** Store Users (B2B/B2C Customers)  
**Last Updated:** January 1, 2026  
**Version:** 1.0

## Overview
The B2Connect Store interface allows customers to browse products, manage their shopping cart, and complete purchases. This guide covers the main features for both B2B and B2C customers, including product search, checkout, and order management.

## Prerequisites
- Valid customer account (B2B or B2C)
- Supported web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- For B2B: Approved company account with credit terms

## Getting Started
1. Navigate to the store URL provided by your administrator
2. Click "Login" in the top right corner
3. Enter your email and password
4. For B2B customers: Select your company from the dropdown if prompted

## Main Features

### Product Browsing and Search
Discover products through various methods

#### How to Use
1. Use the search bar at the top for keyword searches
2. Browse by category using the navigation menu
3. Filter results by price, brand, or specifications
4. Sort by relevance, price, or newest items

![Store Homepage](docs/images/store-homepage.png)

### Shopping Cart Management
Add, remove, and modify items in your cart

#### How to Use
1. Click "Add to Cart" on any product
2. View cart by clicking the cart icon
3. Adjust quantities or remove items
4. Apply discount codes if available
5. Proceed to checkout when ready

#### Screenshots
![Shopping Cart](docs/images/shopping-cart.png)

### Checkout Process
Complete your purchase securely

#### How to Use
1. From cart, click "Proceed to Checkout"
2. Review order summary
3. Select shipping method
4. Choose payment method (invoice for B2B, card for B2C)
5. Enter billing/shipping information
6. Confirm and place order

## Common Tasks

### Task 1: Finding Products
**Purpose:** Locate specific items or browse categories

**Steps:**
1. Navigate to store homepage
2. Use search bar or category navigation
3. Apply filters as needed
4. Click product for details

**Expected Result:** Product listing or detail page displays

### Task 2: Managing Account Information
**Purpose:** Update profile, addresses, or payment methods

**Steps:**
1. Click your name/email in top right
2. Select "My Account"
3. Choose section to update (Profile, Addresses, Payment)
4. Make changes and save

**Expected Result:** Information updated successfully

### Task 3: Viewing Order History
**Purpose:** Check past orders and track current ones

**Steps:**
1. Click "My Account" > "Order History"
2. Use filters to find specific orders
3. Click order number for details
4. Download invoice if needed

**Expected Result:** Order details and status displayed

## Troubleshooting

### Issue 1: Cannot Add Items to Cart
**Symptoms:** "Add to Cart" button not working or showing error
**Cause:** Session expired, item out of stock, or account issue
**Solution:**
1. Refresh the page and try again
2. Check if item is in stock
3. Log out and log back in
4. Contact support if issue persists

### Issue 2: Payment Processing Failed
**Symptoms:** Error during checkout payment
**Cause:** Card declined, insufficient funds, or technical issue
**Solution:**
1. Verify payment information
2. Try alternative payment method
3. Contact your bank if card-related
4. For B2B: Check credit limit with administrator

### Issue 3: Order Not Appearing
**Symptoms:** Order placed but not visible in history
**Cause:** Processing delay or system issue
**Solution:**
1. Wait 15-30 minutes and refresh
2. Check spam/junk email for confirmation
3. Contact support with order details

## Best Practices
- Save frequently used addresses for faster checkout
- Use product filters to narrow down searches
- Review order summary carefully before payment
- Keep login credentials secure
- For B2B: Coordinate with procurement team for approvals
