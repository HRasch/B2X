# Consolidated Reports

Monthly and quarterly analysis of all feedback.

## Monthly Reports

Format: `monthly-report-{YYYY-MM}.md`

**Contents**:
- Total feedback items received
- Summary by type and agent
- Key patterns identified
- Priority 1 items implemented
- Priority 2 items planned
- Metrics (resolution time, implementation rate, etc.)
- Feedback submitted to @process-assistant

**Example**: `monthly-report-2025-12.md`

## Quarterly Trends

Format: `quarterly-trends-{YYYY}-Q{N}.md`

**Contents**:
- Trends across 3 months
- Recurring patterns identified
- Root cause analysis
- Long-term improvements planned
- Comparison: improvement rate quarter-to-quarter

**Example**: `quarterly-trends-2025-Q4.md`

## Created By

@scrum-master - consolidated after each PR merge and monthly consolidation cycle

## Retention

- Keep all monthly reports (historical record)
- Keep all quarterly reports (trend analysis)
- Archive after 1 year to archive/ folder if needed
